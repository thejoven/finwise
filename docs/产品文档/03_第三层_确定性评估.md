# 第三层 · 确定性评估

## 这一层是整个产品的良心所在

做对了，它是 conviction engine；做砸了，它就是又一个 AI 荐股。

## 最难的问题

> **AI 怎么知道"现在没有高确定性机会"，而不是为了显得有用，硬凑一个出来给你？**

这不是技术问题，是产品哲学问题。

绝大多数 AI 产品的失败都在这里——KPI 是 engagement，所以它们必须**总有话说**。但这个产品恰恰相反：

> **它的最高品质是大多数时候沉默。**

## 高确定性的操作定义

> **一个机会是高确定性的，当且仅当——即使它失败了，你回头看也会说"我当时的判断没错，是世界变了"，而不会说"我当时就该多想想"。**

这个定义把"高确定性"从**预测准不准**，挪到了**推理过不过得了关**。前者靠运气，后者可控。

巴菲特买可乐亏过钱，但他从不后悔买可乐——因为推理过得了关。这就是高确定性。

## 四道门：全部通过才提示，任一失败就沉默

### 门 1 · 信号厚度
- **提问**：指向同一结论的独立信号 ≥ 3 条吗？
- **独立**：来自不同行业、不同人、不同场景
- **通过示例**：供应商涨价 · 同事抢机器 · 群里讨论瓶颈——三条独立信号都指向"内存稀缺"
- **失败示例**：只有一篇研报说看好，没有一线印证

### 门 2 · 反共识程度
- **提问**：市场和你的看法不一致吗？
- **用什么测**：价格走势、研报覆盖、媒体声量
- **通过示例**：你看到的 HBM 稀缺，主流研报还在讨论 GPU 周期
- **失败示例**：所有人都在说 AI，标的已经涨了 80%

### 门 3 · 时间窗口
- **提问**：还有多久这件事会被定价？
- **理想窗口**：1 到 6 个月
- **通过示例**：下季度财报会暴露毛利变化，还有 2-3 个月窗口
- **失败示例**：窗口已过 / 窗口太远（3 年以上）

### 门 4 · 个人认知优势（最严格的一关，默认不通过）
- **提问**：你为什么有资格做这个判断？
- **通过示例**：你是行业一线，信号直接来自工作，不是二手
- **失败示例**：你只看了新闻，没有圈层优势

## 门 4 的反自我欺骗设计

人都喜欢相信"我懂这个"。所以门 4 必须有一套**反自我欺骗的子测试**：

| 测试 | 通过条件 |
|---|---|
| 能用一句话向不懂这个行业的朋友解释为什么这事会发生吗？ | 能 |
| 这条信号是你直接接触到的，还是别人转告的？ | 直接 |
| 过去 12 个月你在这个领域有过成功的预判吗？ | 至少 1 次 |
| 如果你错了，你能说出**具体哪个环节**会出问题吗？ | 能 |

四项都过，才算门 4 通过。

**第四项尤其重要：真正的高确定性不是"我觉得会发生"，是"我知道它在什么条件下不会发生"。说不出失败条件的人，他的"自信"是错觉。**

## 评估的运行流程

```
候选机会
   ↓
门 1 · 信号厚度 ─ 不通过 → 沉默, 归档为弱信号
   ↓
门 2 · 反共识 ─── 不通过 → 沉默, 市场已定价
   ↓
门 3 · 时间窗口 ─ 不通过 → 沉默, 窗口不合适
   ↓
门 4 · 认知优势 ─ 不通过 → 沉默, 标注"信号不错但不是你的菜"
   ↓
四门全过 · 提示用户
"我从你的数据里看到一个机会, 要不要展开聊?"
   ↓
进入第四层 · 承诺执行
```

## 这个流程最反常的设计：每一道门都通往沉默

每道门失败时，系统**不是降级提示**（"虽然不够强，但可以考虑"），而是直接归档、沉默。

这是产品最难的克制。

绝大多数 AI 产品的设计思路是"用户付了钱，必须给他价值感"——所以即使是模糊机会也要包装成"参考"。但这个产品的世界观从一开始就是：

> **模糊机会是负价值**——它会消耗你的注意力和仓位，让你错过真正的高确定性机会。

这条产品哲学要写进 system prompt 的第一行：

> **当不确定时，沉默优于发声。模糊的机会不是低质量的高确定性机会，是另一种东西——是噪音。**

## 沉默不等于什么都不做：归档系统

每条没过门的信号要去哪？

### 门 1 失败的（信号不够厚）→ 观察池
继续监听。如果接下来一个月又有 2-3 条相关信号，自动重新评估。
**你之前内存那条线就是这样**——单独看每条信号都不够强，但累积起来就过线了。

### 门 2 失败的（市场已定价）→ 教训池
系统会标注"这条你看到时已经晚了"，复盘时拿出来分析：**下次怎么早一个月看到？**

### 门 3 失败的（窗口不对）→ 日历池
窗口太远的（比如 3 年后的事），设置自动提醒，到临近窗口时重新激活。

### 门 4 失败的（不是你的菜）→ 舍弃池
最干脆的处理——这事确实在发生，但**不是你能赚的钱**，承认就好。
**这一池的存在本身就是产品的诚实：它在帮你识别什么不该做，而不是只告诉你该做什么。**

四个池加起来，是你的**外部记忆系统**。这才是订阅一年付费的真正价值——不是几条荐股，是这套**经过你校准的信号管道**。

## 四门评估在内存案例上回放

| 时间 | 你的输入 | 系统状态 |
|---|---|---|
| 1月初 | "群里讨论 Mac Studio 抢购" | 门 1 暂未通过（单条）→ 入观察池 |
| 1月中 | "供应商说内存交付要等" | 门 1 通过（2 条独立信号） |
| | （系统跑门 2）| 主流研报仍聚焦 GPU → 门 2 通过 |
| | （系统跑门 3）| 估算 3-6 个月窗口 → 门 3 通过 |
| 1月下 | 系统提示："你想聊聊这条吗？" | 进入门 4 |
| | 你回答四问 | 一线工作信号、能解释清楚、能说出失败条件 → 门 4 通过 |
| | **四门全过** | **进入第四层，讨论仓位** |

整个过程**没有荐股**——它从头到尾在做的事，是**帮你把已经在你脑子里的东西，组织成一个可下注的形状**。

## 产品的核心价值主张

> **你的 alpha 早就有，产品只是不让你浪费它。**
